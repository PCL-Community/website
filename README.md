# @pcl-community/website

用于存放 PCL Community 网站的代码仓库。

## 特点

- Vue 3，Vite 6，pnpm - 快速而现代的开发体验
- 手搓组件 - 高度自定义，~~爽不死你~~
- UnoCSS - ~~妈妈再也不用担心我搓不动 CSS 了~~

## 开发

请确保设备上安装了 Node.js 与 pnpm。

```bash
git clone https://github.com/PCL-Community/website.git
cd website
pnpm i
pnpm run dev
```
